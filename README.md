# KPI Dashboard Demo

This project demonstrates how KPI data can be structured, analyzed, and visualized to support performance monitoring and strategic decision-making.

## Objective
To build a simple but scalable KPI dashboard framework that tracks operational performance across multiple branches.

## Dataset
The dataset includes monthly metrics across 3 branches:

- Visits
- Circulation
- Digital Usage
- Program Attendance
- New Memberships
- Customer Satisfaction
- Cost per Visit

## Key Questions This Dashboard Answers
- Which branches are performing best?
- How are engagement and usage trending over time?
- Are we improving cost efficiency?
- How does customer satisfaction align with performance?

## Project Structure
- `data/` → sample KPI dataset
- `dashboard/` → visuals and dashboard outputs
- `notes/` → KPI definitions and logic
